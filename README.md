# Reversi

A reversi app like GUI using ncurses that runs on CUI.


# Installation

### OSX

```
brew tap arks22/reversi
brew install arks22/reversi/reversi
```

### Linux

Plese Download exec file from the [release](https://github.com/arks22/reversi/releases) page.

# Usage

```
$ revesi
```
